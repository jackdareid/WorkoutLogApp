## This is a free to use workout tracking app

#### Visit the website, create a free account using your email address, and voila, you have your very own personal workout tracker and logger.

#### Features:
- Program and workout creation (program's are made of workouts)
- Weight and rep statistic tracking, as well as auto logging based on preferences (not implemented yet)
- Progressive overload setting for auto-logging
- Max weight calculations for each exercise
- Workout reminders (not implemented yet)
- Gemini integration for workout / program creation help (future feature)
- Social aspect (future feature)

##### I'm creating this because I am sick and tired of logging workouts manually on my notes app, and I want to learn express and react better through creating something cool.

#### I hope you enjoy!
